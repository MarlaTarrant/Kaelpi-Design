Zane & Bernice
Bespoke Save the Date invitation and website.

Description

  • Client: Bernice
  • Date: 2018
  • Website: Click to view

Bernice's wedding theme is all things natural, which includes pastel colours and lots of greenery. She wanted a simple and classy one-page wedding website that has everything guests need to plan for the big day.

Inspiration for her website color scheme came from her forest-themed wedding color palette, which we incorporated in the Save the Date Invitation as well. The website includes an information section which then directs the user to a RSVP form where users can successfully RSVP online.
